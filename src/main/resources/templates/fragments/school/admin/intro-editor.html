학교 소개페이지 수정

학교 소개, 기본 정보, 단과대 및 링크 정보를 수정하고 저장할 수 있습니다.

학교 소개

소개 페이지 상단에 노출될 제목과 간단한 설명, 학교의 강점을 입력해주세요.

소개 페이지 상단에 크게 노출되는 제목입니다.

검색 결과나 요약 영역에 함께 노출될 짧은 설명입니다.

예: 높은 취업률, 다양한 교환학생 프로그램 등

기본 정보

학교의 이름, 설립연도, 위치, 간단한 특징을 입력해주세요.

실제 학교 공식 명칭을 입력해 주세요.

연도만 입력하거나 필요 시 연-월 형식으로 입력해도 됩니다.

캠퍼스의 대표 위치(도시, 구 단위)를 간단히 적어주세요.

학교를 한두 키워드로 표현할 수 있는 특징을 적어주세요.

대학 랭킹

내부 정렬이나 노출 순서를 위한 숫자입니다. 숫자가 작을수록 상단에 노출됩니다.

필수가 아니라면 기본값(0 또는 비워두기)으로 두어도 됩니다.

추가 링크 목록

입학 안내, 장학 제도, 기숙사 안내 등 자주 사용하는 페이지 링크를 등록할 수 있습니다.

단과대 정보

여러 단과대를 추가하고, 각 단과대에 포함된 학과들을 입력할 수 있습니다.

홈페이지 URL

대표 국문/영문 홈페이지 주소를 입력합니다.

공식 한국어 홈페이지 주소를 입력해 주세요.

영문 소개 페이지가 없으면 비워두셔도 됩니다.
